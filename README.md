# Astro Basic Project

This is a minimal Astro project scaffold.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm run dev
   ```
3. Open [http://localhost:4321](http://localhost:4321) in your browser.

## Build for production

```
npm run build
```

## Preview production build

```
npm run preview
```
